# Jekyll 本地预览说明

## 当前状态

- **Ruby** 已安装（如 `C:\Ruby32-x64`）
- **依赖** 已通过 `bundle install` 安装（wdm 已跳过，不影响预览）
- 若刚才的终端里 Jekyll 仍在运行，可直接在浏览器打开：**http://localhost:4000** 或 **http://127.0.0.1:4000**

## 以后如何预览

### 方法一：用脚本（推荐）

1. 打开 **PowerShell** 或 **CMD**
2. 进入项目目录：
   ```bash
   cd c:\Users\12072\Downloads\arctanxarc.github.io-master\arctanxarc.github.io-master
   ```
3. 执行：
   ```bash
   .\run-preview.ps1
   ```
   （若提示找不到 Ruby，先关掉当前窗口，新开一个终端再试；或先运行一次 `.\setup-jekyll-preview.ps1`）
4. 在浏览器打开：**http://localhost:4000**
5. 按 **Ctrl+C** 可停止预览

### 方法二：手动命令

```bash
cd c:\Users\12072\Downloads\arctanxarc.github.io-master\arctanxarc.github.io-master
bundle exec jekyll serve
```

然后在浏览器打开 **http://localhost:4000**。

## 首次安装（仅需一次）

若在新电脑或新系统上、还未装过 Ruby，请：

1. **以管理员身份** 打开 PowerShell
2. 进入项目目录后执行：`.\setup-jekyll-preview.ps1`
3. 若安装完成后脚本没有自动启动预览，请关闭该窗口，**新开一个** PowerShell，再执行 `.\run-preview.ps1`

## 已做修改

- **Gemfile**：已注释掉 `wdm`（在部分 Windows 上编译失败），并增加 `tzinfo-data`（Windows 时区数据），以便 Jekyll 在本地正常跑起来。
