# 部署到线上 (GitHub Pages)

远程仓库：**https://github.com/liouqihuei/website1.git**

## 一、在电脑上执行（任选一种方式）

### 方式 1：用脚本（推荐）

在 **PowerShell** 中进入项目目录，执行：

```powershell
cd "c:\Users\12072\Downloads\arctanxarc.github.io-master\arctanxarc.github.io-master"
.\deploy.ps1
```

### 方式 2：手动输入命令

在项目目录下打开 **PowerShell**、**CMD** 或 **Git Bash**，依次执行：

```bash
cd "c:\Users\12072\Downloads\arctanxarc.github.io-master\arctanxarc.github.io-master"

# 若尚未初始化
git init
git remote add origin https://github.com/liouqihuei/website1.git

# 提交并推送
git add .
git commit -m "部署个人网站"
git branch -M main
git push -u origin main
```

若之前已经添加过 `origin`，可先查看并（如需）修改：

```bash
git remote -v
git remote set-url origin https://github.com/liouqihuei/website1.git
```

---

## 二、在 GitHub 上开启 Pages

1. 打开：https://github.com/liouqihuei/website1  
2. 进入 **Settings** → **Pages**  
3. **Source** 选择：**GitHub Actions**（不要选 “Deploy from a branch”）  
4. 保存后，每次推送到 `main` 或 `master` 分支会自动构建并部署。

---

## 三、访问地址

部署成功后，站点地址为：

**https://liouqihuei.github.io/website1/**

首次推送或修改后，通常 1–2 分钟内可访问；可在仓库 **Actions** 页查看构建进度。
